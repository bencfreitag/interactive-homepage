**Interactive Homepage**

This project is an interactive personal homepage originally built while teaching Fundamentals of Computer Science using CodeHS. It combines HTML, CSS, JavaScript, and jQuery to create a playful collection of interactive elements, small games, animations, and randomly generated maze activities.

**Features**
- Interactive homepage layout built with HTML and CSS
- JavaScript and jQuery-based user interactions
- Randomly generated maze activities
- Simple browser-based games and animations
- Creative elements and visual experiments

**Technologies Used**
- HTML
- CSS
- JavaScript
- jQuery
- Project Context

This project was originally developed in CodeHS as part of my teaching and programming practice. It has been migrated to GitHub as a portfolio project to demonstrate frontend fundamentals, interactive UI logic, creative problem-solving, and browser-based JavaScript programming.
